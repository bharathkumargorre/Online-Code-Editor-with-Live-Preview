# 💻 Online Code Editor with Live Preview

An interactive web-based code editor that allows users to write HTML, CSS, and JavaScript and see the output instantly in real-time.

---

## 🚀 Features

- ✨ Real-time Live Preview
- 🧑‍💻 Write HTML, CSS, JavaScript in one place
- ⚡ Instant output rendering
- 🎨 Clean and responsive UI
- 🔄 Auto-updating preview window
- 📱 Beginner-friendly interface

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- React (if used)
- Vite

---

## 📂 Project Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/Online-Code-Editor-with-Live-Preview.git
````

### 2. Go to project folder

```bash
cd Online-Code-Editor-with-Live-Preview
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```
